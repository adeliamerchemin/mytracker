Setup Guide

Connect FinTrack to Google Sheets in 5 easy steps

Step 1 — Create Google Sheet

Go to sheets.google.com and create a new spreadsheet. Copy the Spreadsheet ID from the URL:
https://docs.google.com/spreadsheets/d/YOUR_ID_HERE/edit

Step 2 — Open Apps Script

In your Google Sheet → Extensions → Apps Script.
Delete all existing code. Paste the entire contents of appscript.gs.
Replace YOUR_SPREADSHEET_ID_HERE with your actual Sheet ID.

Step 3 — Run Setup Once

In the Apps Script editor, select the function setupSpreadsheet from the dropdown and click ▶ Run. This creates all 12 month tabs automatically.

Step 4 — Deploy as Web App

Click Deploy → New Deployment → Web App.
Set Execute as: Me and Who has access: Anyone.
Click Deploy and copy the Web App URL.

Step 5 — Connect Frontend

Open script.js and replace:
const APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
…with your actual Web App URL. Save and reload the page. Done! 🎉
